# Monetization, Communities, Maintenance, Final Note

## If You Run Short on Time

Drop priority order:

1. **Stretch sections** in every phase — skip first.
2. **Project A** (Phase 1 landing page) — reducible to a single page in 2h. Task 1.4 (subagent) is what matters.
3. **Task 2.4** (claude-squad on a real refactor) — skip if Project B ran long.
4. **Task 5.3** (community orchestrator) — reducible to a 1-hour read-through if behind.
5. **Anthropic Academy courses** — the ones marked "optional" or "stretch" can be skipped. Sub-agents and MCP courses are not optional.
6. **[Project D](05-phase-4-mcp.md)** (MCP server) — non-negotiable. Highest-ROI thing in the program.
7. **[Project E](06-phase-5-multi-agent.md)** (digest bot) — non-negotiable. Locked.
8. **[Capstone](07-phase-6-capstone.md)** — non-negotiable. Ship it.

If you're consistently under 4 hrs/week, extend to 28–30 weeks rather than compress depth.

## Monetization Paths

1. **MCP-as-a-service** — Phase 4 MCP, hosted, managed auth. $5–$15/mo to power users.
2. **Niche SaaS** — your capstone with one-time payment ($29–$79 lifetime) or low monthly fee.
3. **Plugin bundles** — Phase 3 plugin sold as premium via Gumroad or Lemonsqueezy.
4. **Paid templates** — capstone repo as a starter ($49–$99 one-time) for forks.
5. **Consulting** — once shipped, "I help engineering teams build multi-agent systems with Claude Code" is a real position. $500–$2000/engagement to start.
6. **GitHub Sponsors** — real money for solo maintainers of useful MCPs and orchestration tooling.

## Maintenance Mode (after Week 24)

Pick two:
- Ship v2 of the capstone from user feedback.
- Write a long-form blog post or talk on your 24-week journey.
- Contribute one PR to `obra/superpowers`, `gsd-build/get-shit-done`, `modelcontextprotocol/servers`, or an Anthropic official plugin.
- Mentor someone else through this program.
- Open Project E (digest bot) as a paid niche newsletter.

## Final Note

The frontier-model race has quieted relative to the harness race. Codex vs Claude Code is now a more meaningful contest than GPT vs Opus. The scaffolding around the model is where the work — and the money — is.

By Week 24 you'll know that scaffolding from the inside. You'll be one of a small group of engineers who can think in coordinator-and-specialists rather than prompt-and-response. That gap is real and currently widening.

Ship in public, write what you learn, don't stack orchestrators. See you on the other side.
